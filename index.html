CSS

  • Enter your CSS notes here
  • A margin indicates the space want to block off around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.
  • The period before a word designates this as a class selector. Any HTML element assigned to a class in HTML will gain assigned CSS properties.
  • A "link rel="stylesheet" href="insert CSS file location here" /" bracketed with opening and closing tags is used to create an external style stylesheet.
  • Declarations are made of two components. Example, Header{width: 100%; height: 120px;} we have properties(width, height) and their values(100%, 120 px)

javascript

  • Enter your javascript notes here
  • Variables are containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable. For example "let myvariable;" using a semicolon to ends the statment.
  • A String variable is a sequence of text denoted as such by enclosing the text in between single/double qoute marks.
  • A number variable is a number with no quotes.
  • A boolean is a true/false variable. No quotes required.
  • An Array is a structure that allows for the storing of multiple values.
  • Everything in JavaScript is an Object and can be stored in a variable
  • JS typically executes code top to bottom but certain conditionals such as if statments allow us to interupt control flow.s
